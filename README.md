# Task Manager API
A RESTful API built with Python and Flask to manage user tasks efficiently.

## Features
* **CRUD Operations**: Complete support for creating, reading, updating, and deleting tasks.
* **Scalable Architecture**: Built with modularity and clean code principles to ensure maintainability.
* **DevOps Integration**: Includes Docker support for easy containerized deployment.

## Tech Stack
* Python
* Flask
* SQLAlchemy
* Docker

## How to Run
1. Clone this repository: `git clone https://github.com/AshuShrma/task-manager-api.git`
2. Install the required dependencies: `pip install -r requirements.txt`
3. Start the application: `python app.py`
